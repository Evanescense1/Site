1 Install XAMPP
2 To properly execute XAMPP you need to open the properties xampp-control.exe ->
compatibility -> checked run this program as an administrator
3 Open the XAMPP control panel and run apache and mysql
4 Extract the files to C:\xampp\htdocs
5 Open the link localhost/phpmyadmin
6 Click create in the navigation sidebar
7 Give the database name as (site) click on the create button
8 After creating the database name, click import
9 View the file in the [site/database/appointment.sql] directory]
10 Open any browser and enter http://localhost/site
